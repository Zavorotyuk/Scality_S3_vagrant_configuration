 ### To run s3

```shell
git clone https://github.com/Zavorotyuk/Scality_S3_vagrant_configuration.git
```
```shell
cd Scality_S3_vagrant_configuration
```
```shell
vagrant up
```
```shell
vagrant ssh
```
```shell
cd S3/
```
```
sudo npm start
```
